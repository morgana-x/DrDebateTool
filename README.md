# DrDebateTool

+ kokoro
+ nonstop
+ hanron
  
For further documentation on what each value means: (Applies to DR1 as well)
  + please see https://github.com/BitesizeBird/Danganronpa-Modding's
  +  `DR2 DAT OPCODE EXPLANATIONS`
